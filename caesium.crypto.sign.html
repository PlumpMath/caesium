generate-signing-keys

(generate-signing-keys)(generate-signing-keys seed)

Generate a public-key and secret-key for signing with crypto_sign_ed25519_seed_keypair. If a seed is not provided, one is taken from randombytes.

A map of the secret seed and public-key is returned.

verify

(verify pk sm)(verify pk msg sig)

Verify a signed message or a message and a detached signature.

When given a valid signed message, returns the unsigned message. When given a valid signature, returns nil. When given an invalid signed message or signature, raises RuntimeException.
